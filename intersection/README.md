# Dependancies

There is a dependancy: gnuplot. It can be easily installed with:

On MacOS:

	brew install gnuplot
	
On Ubuntu:

	sudo apt install gnuplot
	
# Programm execution
	
Run this programm as:

	g++ intersection.cpp -std=c++11 && ./a.out
